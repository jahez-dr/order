خطوات الربط التفصيلية

1

افتح Google Sheet الخاص بك

تأكد أن الصف الأول يحتوي على العناوين: First name Phone City Product name price Order ID Order date SKU

2

افتح محرر Apps Script

من القائمة: الإضافات ← Apps Script ← احذف الكود الموجود والصق هذا:

function doGet(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheets()[0]; var data = sheet.getDataRange().getValues(); var headers = data[0]; var rows = []; for (var i = 1; i < data.length; i++) { if (!data[i][0]) continue; // skip empty rows var row = {}; for (var j = 0; j < headers.length; j++) { row[String(headers[j]).trim()] = data[i][j]; } rows.push(row); } var output = ContentService.createTextOutput(JSON.stringify({ success: true, count: rows.length, data: rows, timestamp: new Date().toISOString() })); output.setMimeType(ContentService.MimeType.JSON); return output; } function doPost(e) { // للتحديث من النظام إلى الشيت (اختياري) var body = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheets()[0]; var data = sheet.getDataRange().getValues(); for (var i = 1; i < data.length; i++) { if (String(data[i][5]) === String(body.orderId)) { sheet.getRange(i+1, 5).setValue(body.status); break; } } return ContentService.createTextOutput(JSON.stringify({success: true})) .setMimeType(ContentService.MimeType.JSON); }
3

انشر كـ Web App

اضغط نشر ← نشر جديد ← اختر نوع تطبيق ويب ← في "من يمكنه الوصول" اختر أي شخص ← انسخ الرابط

4

الصق الرابط في الحقل أعلاه واضغط ربط

الرابط يبدو هكذا: https://script.google.com/macros/s/.../exec

Zapier — الأكثر سهولة

اربط أي نموذج (Google Forms, Typeform, Facebook Leads) مع النظام بدون كود. كل طلب جديد يُرسل تلقائياً.

مجاني جزئياً لا يحتاج كود
1

افتح zapier.com وأنشئ "Zap" جديد

اختر Trigger: مصدر طلباتك (Google Sheets / Facebook Lead Ads / Typeform / WooCommerce)

2

أضف Action: Webhooks by Zapier

اختر "POST" واملأ URL بـ: https://yoursite.com/api/orders

3

حدد الحقول

name, phone, city, product, price, sku — وادخلها من بيانات المصدر

API / Webhook المباشر

أرسل الطلبات مباشرة من موقعك أو متجرك إلى النظام عبر HTTP POST.

مثال Fetch Request:

fetch('YOUR_WEBHOOK_URL', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: "أحمد ولد محمد", phone: "46123456", city: "نواكشوط", product: "معجون مبيض الأسنان", price: 590, sku: "5712", orderId: Date.now() }) });

WooCommerce

استخدم إضافة "WooCommerce Zapier" أو "Order Webhooks for WooCommerce" لإرسال كل طلب جديد مباشرة.

Shopify

من إعدادات Shopify → Webhooks → أضف webhook لحدث "Orders created" مع الرابط أعلاه.

استيراد ملف CSV

حمّل ملف CSV يحتوي على طلباتك. الأعمدة المطلوبة: name, phone, city, product, price, sku
